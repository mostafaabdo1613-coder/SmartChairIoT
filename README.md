# SmartChair v3 — النظام الكامل (Hardware + Backend + Push)

## 📁 الملفات الجديدة المهمة
```
smartchair_v2/
├── hardware/
│   ├── esp32_smartchair.ino      ← كود ESP32 (نظام تأكيد النبض 3 ثواني)
│   ├── uno_chair_controller.ino  ← كود الأونو (+ بادري تشغيل)
│   └── helmet_nano.ino           ← كود الخوذة (+ سقوط + إغماء)
├── app.py                        ← Flask + جدولة المواعيد سيرفر-سايد
├── static/sw.js                  ← Service Worker للـ Push
├── static/push.js                ← تفعيل إشعارات الموبايل
└── requirements.txt
```

## 🔧 الخطوات بالترتيب

### 1. ارفع أكواد الـ Hardware
- `hardware/esp32_smartchair.ino` → على الـ ESP32
- `hardware/uno_chair_controller.ino` → على الأونو
- `hardware/helmet_nano.ino` → على النانو (الخوذة)

### 2. تأكد إن قواعد Firebase لسه شغالة
لو الموقع بيديك خطأ "Permission Denied" مع Firebase، يبقى الـ Test Mode انتهت
صلاحيته (بتنتهي تلقائي بعد 30 يوم). افتح Firebase Console > Realtime Database
> Rules، وحط:
```json
{
  "rules": {
    ".read": true,
    ".write": true
  }
}
```
⚠️ ده مفتوح للجميع (مناسب للتطوير فقط، مش للإنتاج الحقيقي).

### 3. فعّل Push Notifications للموبايل (اختياري لكن مهم)
```bash
pip install py-vapid
vapid --gen
```
هيطلعلك ملفين مفاتيح، افتحهم وانسخ الـ private/public key وحطهم في
`app.py` في أول الملف:
```python
VAPID_PRIVATE_KEY = "..."
VAPID_PUBLIC_KEY  = "..."
```
لو سبتهم فاضيين، الموقع هيشتغل عادي بس من غير إشعارات موبايل (الصوت في
الكرسي والتنبيهات جوه الموقع هتفضل شغالة عادي).

### 4. شغّل المشروع
```bash
pip install -r requirements.txt
python app.py
```
افتح الموقع، اضغط "فعّل إشعارات الموبايل" من أي صفحة، ووافق على الإذن.

## ✅ إيه اللي اتظبط

| المشكلة | الحل |
|---|---|
| النبض بيفضل عالق على آخر رقم لما تفصل السنسور | `system/last_seen` + فحص الجهاز كل ٢ث في الموقع |
| النبض صفر/غير طبيعي مش بيعمل إنذار فوري | نظام تأكيد ٣ ثواني في ESP32 قبل أي إنذار (منع الإنذار الكاذب) |
| السقوط مش متفرق عن الإغماء | الخوذة بقت تكتشف الاتنين: صدمة مفاجئة (X) أو ثبات تام 60 ثانية (W) |
| مواعيد الدواء مش بترن لو الموقع مقفول | الفحص بقى سيرفر-سايد (APScheduler في Flask) مش في المتصفح |
| مفيش إشعار يوصل للموبايل | Web Push كامل (service worker + VAPID) |
| الأونو ميعملش حاجة عند التشغيل | بادري بسيط (3 نغمات) في `setup()` |

## ملاحظة عن نظام النبض
- لو شِلت إصبعك من الحساس: bpm = 0 فوراً، لكن الإنذار (الصوت + التنبيه)
  بيتأخر **3 ثواني بالظبط** قبل ما يتفعل، عشان نتجنب إنذار كاذب من حركة
  بسيطة. لو الـ 3 ثواني عدت والنبض لسه صفر أو غير طبيعي، هيتفعل الإنذار
  ويوصل تنبيه على الموقع + بازر في الكرسي (لو سويتش "تفعيل نظام الأمان"
  شغال من لوحة التحكم).
